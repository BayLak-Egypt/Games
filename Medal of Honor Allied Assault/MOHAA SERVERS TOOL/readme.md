# 🎮 MOHAA SERVERS TOOL v1.0

An integrated tool for managing and monitoring **Medal of Honor** series game servers. This tool is designed to provide full control and facilitate administrative and technical tasks for users.

## 🛡️ Supported Games
The tool supports the classic series:
*   **MOHAA** (Medal of Honor: Allied Assault)
*   **MOHSH** (Medal of Honor: Spearhead)
*   **MOHBT** (Medal of Honor: Breakthrough)

---

## 🚀 Key Features

The tool includes a menu featuring the following powerful utilities:

### 1. Start Menu
A dedicated menu containing classic hacks and tweaks, such as **No Recoil**, **Third Person**, and other legacy game commands.

### 2. Server Management Tools
| Feature | Description |
| :--- | :--- |
| **Status** | Get detailed server information (Country, player list, etc.). |
| **Bruteforce** | Test RCON passwords. (Requires a wordlist like [rockyou.txt](https://github.com/zacheller/rockyou/raw/refs/heads/master/rockyou.txt.tar.gz)). |
| **Scan File** | Scan RCON paths and retrieve comprehensive info about server directories. |
| **Add IP** | Manually add any IP (including Localhost). |
| **Remove** | Remove any IP from the list. |
| **Pining** | Pin/Unpin favorite servers for quick access. |
| **Blocked** | Block specific IPs and move them to a blacklist. |
| **Refresh** | Fully refresh the server list. |
| **Setting** | Customize themes, language, and more. |
